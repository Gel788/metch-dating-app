## 🎭 КОМПОНЕНТЫ И СТИЛИ

### `globals.css` - Главные стили:

#### Градиенты:
```css
.gradient-dark - темный фон всего приложения
.gradient-gold - золотой для Premium
.gradient-purple - фиолетовый
.gradient-pink - розовый
.text-gradient - текст с градиентом
```

#### Анимации:
```css
.animate-fadeIn - плавное появление
.animate-slideIn - появление снизу
.animate-scaleIn - увеличение
.animate-glow - свечение (пульсация)
```

#### Компоненты:
```css
.card - основная карточка с blur
.card-hover - с эффектом hover
.card-premium - с золотой обводкой
.btn-primary - розовая кнопка
.btn-secondary - серая кнопка
.btn-gold - золотая для Premium
.btn-purple - фиолетовая
.input - поле ввода
.badge - бейдж
.glass - эффект стекла
```

## 🛠️ ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Использовано:
- ✅ **TailwindCSS** - utility-first CSS
- ✅ **Custom CSS** в `globals.css`
- ✅ **CSS Animations** (keyframes)
- ✅ **Backdrop Filter** (blur)
- ✅ **CSS Gradients** (linear, radial)
- ✅ **Transform** (translate, scale, rotate)
- ✅ **Transitions** (all 0.3s ease)

### Кастомный скроллбар:
```css
::-webkit-scrollbar - 10px ширина
::-webkit-scrollbar-thumb - gradient pink→purple
::-webkit-scrollbar-track - темный фон
```

### Выделение текста:
```css
::selection - pink/30% с белым текстом
```
